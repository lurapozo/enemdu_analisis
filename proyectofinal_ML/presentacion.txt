completion: 
Numpy es una biblioteca de Python que proporciona herramientas para trabajar con arreglos de datos. Estos arreglos son estructuras de datos que contienen valores homogéneos y se usan para almacenar información. 

Numpy ofrece tres tipos principales de funciones para manipular los arreglos: funciones de Arreglos, funciones de Manipulación de Arreglos, y funciones de Operaciones Matemáticas. 

Las Funciones de Arreglos permiten crear nuevos arreglos a partir de otros, copiarlos, cambiar su tamaño, rellenarlos con valores específicos, entre otros. 

Las Funciones de Manipulación permiten reorganizar el contenido dentro del arreglo como ordenar los elementos, girar el contenido del mismo e invertir su disposición.
